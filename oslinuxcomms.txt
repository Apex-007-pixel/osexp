1.   Apply awk command on suitable data and describe commands.



cat > student.txt

1 Rahul 70
2 Amul  89
3 Neha  99
4 Leon   90

CTRL + D (Saved)


Check File:

cat student.txt

O/p = data




// Commands Execution now begins.....

Print all records:

awk '{print}' student.txt

Print names only:

awk '{print $2}' student.txt

Print marks > 90:

awk '$3>90 {print $2,$3}' student.txt

Explanation
$1 = ID
$2 = Name
$3 = Marks

5. Demonstrate and describe chmod, adduser, pwd commands. 



A   chmod
Important:

chmod needs an existing file or folder.


So 

First create file:

touch demo.txt

Then:

chmod 777 demo.txt

After chmod:

ls -l demo.txt



B.   adduser


sudo adduser student1


C.  pwd

pwd




Explanation:
chmod = permission
adduser = create user
pwd = current path

6. Demonstrate and describe chmod and chown commands. 





A.   chmod


chmod needs an existing file or folder.


So 

First create file:

touch demo.txt

Then:

chmod 777 demo.txt




B.   chown



Additional Setup :


1. demo.txt must exist

Create if needed:

touch demo.txt


2. user1 must exist

Create:

sudo adduser user1


3. check

ls -l demo.txt


Now actual Command Execution :


sudo chown user1 demo.txt

ls -l demo.txt

Explanation:
chmod = permissions
chown = owner
